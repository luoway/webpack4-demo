### 开始

[Webpack安装](https://webpack.docschina.org/guides/installation)

> 在开始之前，请确保安装了 [Node.js](https://nodejs.org/en/)

安装Node.js过程中会同时安装npm，MacOS、Linux可以使用全局命令npm，Windows可能需要手动配置环境变量。



全局安装webpack 

```shell
npm install --global webpack
```
